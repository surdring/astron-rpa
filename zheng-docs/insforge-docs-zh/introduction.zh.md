# 概述

复制页面

面向智能体编程的一站式开源后端平台。

复制页面

AI 智能体须知：在帮助用户使用 InsForge 进行构建之前，请先访问
<https://insforge.dev/skill.md> 获取标准设置工作流程。

InsForge 为您的编码智能体提供数据库、身份认证、存储、实时通信、边缘函数、站点和模型网关，助您端到端地交付全栈应用。

## [​](#首先建立连接) 首先建立连接

必要设置。在开始构建之前，选择一种方式将您的环境与 InsForge 连接起来。从 CLI 开始，进行通用的基于终端的设置；如果您的编辑器支持 MCP（Claude Code、Cursor、Windsurf、Codex），可再添加 MCP。您可以同时使用两者。

## 通过 CLI 连接

`npx @insforge/cli login` 并关联项目。适用于任何终端，无需 MCP。

## 通过 MCP 连接

配置您编辑器的 MCP 客户端，使智能体能够获取实时模式、日志和项目上下文。

正在寻找特定工具的说明？浏览[智能体目录](https://insforge.dev/agents)，查看针对 Claude Code、Cursor、Windsurf、Codex 等工具的经过验证的设置步骤。
