## T1. 认证与启动

### T1.1 启动页加载

| 项目 | 内容 |
|------|------|
| **前置条件** | 前端 `pnpm dev:web` 已启动，后端 `172.16.100.211:7130` 可访问 |
| **测试步骤** | 1. 打开浏览器访问 `http://localhost:1420/boot.html` |
|  | 2. 观察启动页是否正常渲染 |
|  | 3. 检查是否有启动进度条动画 |
|  | 4. 检查浏览器控制台是否有报错 |

| **验证项** | □ 启动页渲染正常 □ 无 401/403/500 错误 |

### T1.2 用户注册

| 项目 | 内容 |
|------|------|
| **前置条件** | 启动页已加载，用户未登录 |
| **测试步骤** | 1. 在启动页点击「注册」切换至注册表单 |
|  | 2. 输入邮箱地址（如 `test@test.com`） |
