### 修复过程

Edge Functions 部署后遇到 "Security Initialization Error"，根因为 Deno 2.9.1 中 `globalThis.Deno` 不可配置（non-configurable），`worker-template.js` 尝试用 `Object.defineProperty` 重新定义导致错误。解决方案：

1. 修改 `InsForge/functions/worker-template.js`：不再重新定义 `globalThis.Deno`，改为直接 patch `Deno.env` 方法
2. 修改 notify 函数代码：移除 `client.auth.getCurrentUser()` 调用，改用 `atob()` 从 JWT 解析 `payload.sub` 获取用户 ID
3. 创建 `notification_logs` 和 `blacklist` 表，配置 RLS 策略
4. 刷新 PostgREST schema 缓存
